Creating tour requests

If you are in the Ordinary tab and you have clicked on the CREATE REQUEST button, you can create a request for an ordinary tour by entering the required fields. If all fields are validly filled (if not, you will see a message below the fields), you can create the tour request by clicking the CONFIRM button.

Below is a correctly completed creating request form for ordinary request




If you are in the Complex tab, you can create a complex tour. A complex tour consists of multiple ordinary tours. By filling in the fields as for an ordinary tour and clicking ADD, you have added one ordinary tour to the complex request. You can add as many as you want, and they will be visible in the table below the creation form. By clicking the CONFIRM button, you have successfully created a request for a complex tour.
