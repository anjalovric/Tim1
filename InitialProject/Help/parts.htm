Request parts

Here, you can see all parts of selected complex request. You can see name and status of a ordinary request that is part of a complex request.
ACCEPTED: if the request has been accepted.
ON WAITING: if the request is still pending.
INVALID: if the request has not been accepted, and the deadline has passed.

You can also see the details about accepted ordinary tour request. If status is invalid or waiting, you will get a message that it is not possible to see the details.
