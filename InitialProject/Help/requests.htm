Requests

Here, you can see all your requests for both regular and complex tours. The difference between these two types is that a complex tour consists of multiple regular tours. In the Ordinary tab, only the requests marked as regular will be displayed, excluding those that are part of a complex tour. Besides the name of the tour, you can also see the status of each request.
In the Ordinary tab, the status of a request can be one of the following: ACCEPTED: if the request has been accepted. ON WAITING: if the request is still pending. INVALID: if the request has not been accepted, and the deadline has passed. In the Complex tab, you will find the complex tours and their status. The status follows the same criteria as for regular tours, but for a complex tour to have the status ACCEPTED, all its parts must be accepted. You also have the Request parts field, which allows you to view the individual parts of a complex tour. If a part has been accepted, you will be able to see the scheduled start time set by the guide. Otherwise, you will see a message indicating that the request is still pending.
